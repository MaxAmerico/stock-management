# Electronic Store Inventory System
A lightweight and interactive Command Line Interface (CLI) application built with Python to manage stock, sales data, and staff records for an electronics retail store.

## 🚀 Overview
This project was developed to automate the basic operations of an electronics store. It features a secure access system with different permission levels, allowing employees to manage stock and administrators to access financial performance data and human resources.

## ✨ Key Features
Secure Access: Multi-level authentication for General Staff and Administrators.

Inventory Management: Real-time adding and removing of products.

Financial Insights: Monthly average sales calculator and annual total revenue tracking.

HR Management: Exclusive area for administrators to hire or dismiss employees.

Input Validation: User-friendly interface with data normalization (case-insensitive inputs).

## 🛠️ Technologies
Language: Python 3.x

Paradigm: Procedural Programming
